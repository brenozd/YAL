# YAL - Yet Another Language
____
### What is this?
This is a simple Scanner and Parser made to Compilers course in UNIFEI.

Keep in mind that this isn't a professional project and has learning purposes only!

### What was used?
In this project we used:
* GCC 9.3.0
* Flex 2.6.4
* Bison 3.5.1

### How to use?
Just type
> make

to build and run.
